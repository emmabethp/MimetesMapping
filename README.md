# Mimetes Mapping
Repo to explore mapping _Mimetes fimbriifolius_ and _Leucospermum conocarpodendron_ individuals and their mortality at Silvermine using BioSCape data

See `code/aviris_mesma_workflow.R` for a commented AVIRIS-NG workflow using `terra` and `luna` (read, crop, vegetation indices, point extraction, MESMA).
